# 2026_! - Professional Gaming Storefront

Chào mừng bạn đến với **2026_!** - Một nền tảng thương mại điện tử chuyên nghiệp dành cho linh kiện máy tính và thiết bị gaming cao cấp.

## 🚀 Tính năng nổi bật
- **Giao diện Gaming Modern**: Thiết kế đậm chất game thủ với hiệu ứng Neon và Glassmorphism.
- **Quản lý kho hàng (Admin)**: Hệ thống quản lý sản phẩm mạnh mẽ với Dashboard thống kê doanh thu.
- **Đồng bộ SQL thông minh**: Tự động nạp dữ liệu từ file `.sql` gốc vào hệ thống web.
- **Hệ thống Tin tức**: Cập nhật những xu hướng công nghệ mới nhất.
- **Báo cáo định kỳ**: Tự động gửi báo cáo doanh thu hàng tháng qua Email.

## 🛠️ Công nghệ sử dụng
- **Backend**: Python Flask
- **Database**: SQLite (Tương thích T-SQL)
- **Frontend**: TailwindCSS, JavaScript (Vanilla)

## 📦 Hướng dẫn cài đặt
1. Clone dự án:
   ```bash
   git clone https://github.com/dinhtrongmanhquan/2026.git
   ```
2. Cài đặt thư viện:
   ```bash
   pip install -r requirements.txt
   ```
3. Chạy ứng dụng:
   ```bash
   python app.py
   ```
